# Excel 변환기 실행 방법

## 1단계 – 라이브러리 설치 (최초 1회만)

터미널(Terminal)을 열고 아래 명령어를 실행하세요.

```bash
cd ~/Documents/shipping/excel-converter
pip install -r requirements.txt
```

> Mac에서 pip3를 써야 할 경우:
> `pip3 install -r requirements.txt`

---

## 2단계 – 서버 실행

```bash
python app.py
```

> Mac에서 python3를 써야 할 경우:
> `python3 app.py`

아래 메시지가 나오면 정상 실행된 것입니다:
```
==================================================
  Excel 변환기 서버 시작
  브라우저에서 http://localhost:5000 열기
  종료: Ctrl+C
==================================================
```

---

## 3단계 – 브라우저에서 사용

1. 브라우저(Chrome, Safari 등)를 열고 **http://localhost:5000** 접속
2. Before 형식의 `.xlsx` 파일 업로드
3. **변환 후 다운로드** 버튼 클릭
4. 변환된 파일이 자동으로 다운로드됩니다 (`원본파일명_converted.xlsx`)

---

## 변환 내용

| 항목 | Before | After |
|------|--------|-------|
| 날짜 형식 (DELI_DATE, CFMD_DATE) | `2026-03-24 00:00:00` | `2026. 3. 24` |
| 행 분리 (Pallet No 등) | 한 행에 `;`으로 구분된 복수 값 | 각 값을 별도 행으로 분리 |

분리 대상 컬럼: `Pallet No`, `BATCH PROPOSAL`, `MOLD REQUEST`, `Prod Date`

---

## 종료 방법

터미널에서 **Ctrl + C** 를 누르면 서버가 종료됩니다.
