☁️ Cloud Setup / 雲端設定

To share data across multiple users and devices, connect Firebase (free).
Currently running in Local Mode — data is saved only on this device.

📋 Setup Steps / 設定步驟
  1. Go to console.firebase.google.com and sign in (free Google account)
  2. Click Add project → Enter project name → Create
  3. Click </> Web icon → Register app → Copy the firebaseConfig object
  4. In the left menu: Build → Realtime Database → Create database → Choose region → Start in test mode
  5. Open this HTML file in a text editor, find FIREBASE_CONFIG near line 600, replace the placeholder values with your copied config
  6. Host the file on app.netlify.com/drop (free — just drag & drop)
  7. Share the Netlify URL with your team!
⚠️ Security Note: "Test mode" allows anyone with the URL to read/write data for 30 days. For production use, add Firebase Authentication rules.
